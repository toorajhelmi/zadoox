# Post-MVP Features

This document outlines advanced features and enhancements planned for Zadoox beyond the MVP scope. These features will be implemented after the core MVP is complete and validated with users.

---

## Advanced Editing Features

### Intelligent Editing & Productivity

- [ ] **Advanced search & navigation**:
  - [ ] Fuzzy find (like VS Code Cmd+P)
  - [ ] Search in document (with preview)
  - [ ] Replace with regex support
  - [ ] Find references/definitions
  - [ ] Navigate by heading structure
  - [ ] Bookmark system

- [ ] **Selection & manipulation**:
  - [ ] Select by paragraph/section
  - [ ] Multi-cursor editing
  - [ ] Column selection (Alt+drag)
  - [ ] Expand/contract selection (word → sentence → paragraph)
  - [ ] Smart cut/copy (preserve formatting)
  - [ ] Paste with format options

- [ ] **Smart formatting**:
  - [ ] Auto-format on paste
  - [ ] Format normalization (consistent spacing, quotes)
  - [ ] Auto-capitalization for sentences
  - [ ] Smart lists (auto-continue numbering)
  - [ ] Auto-link detection (URLs, emails)
  - [ ] Markdown shortcut expansion

- [ ] **Content structure tools**:
  - [ ] Smart headings (auto-level detection)
  - [ ] Outline view with drag-and-drop reordering
  - [ ] Section folding/collapsing
  - [ ] Mini-map (code editor style)
  - [ ] Breadcrumb navigation for nested sections
  - [ ] Jump to section/symbol

### Templates & Snippets

- [ ] **Code snippets** (like VS Code):
  - [ ] Text templates (for common structures)
  - [ ] Custom snippets
  - [ ] Snippet variables
  - [ ] Template library
  - [ ] Project-specific templates

### Writing Modes & Aids

- [ ] **AI writing modes**:
  - [ ] Ideation mode (free-form, no constraints)
  - [ ] Draft mode (rough writing allowed)
  - [ ] Fragment mode (quick notes, bullet points)
  - [ ] Refinement mode (focus on polishing)
  - [ ] Mode indicators and switching

- [ ] **Writing aids**:
  - [ ] Focus mode (hide everything but current paragraph)
  - [ ] Distraction-free writing
  - [ ] Typewriter mode (scroll keeps cursor centered)
  - [ ] Reading mode (large text, optimized for reading)
  - [ ] Side-by-side comparison (before/after AI improvements)

### Keyboard Shortcuts & Command Palette

- [ ] **Comprehensive keyboard shortcuts**:
  - [ ] Customizable keybindings
  - [ ] Command palette (Cmd+K / Ctrl+K)
  - [ ] Quick actions menu
  - [ ] Macros/automation support
  - [ ] Shortcut hints/cheat sheet
