# Zadoox

**Zadoox** — Transform your thoughts and ideas into elegant code and documentation.

An AI-powered documentation platform that understands your code. Zadoox seamlessly connects your codebase with your documentation, enabling bidirectional links between code and docs while providing intelligent writing assistance.

## What is Zadoox?

Zadoox is a hybrid documentation platform that operates as both:
- **Web Application**: Full-featured documentation editor with AI assistance, collaboration, and multi-format export
- **Cursor IDE Integration**: Code-documentation linking and inline documentation features directly in your editor

Zadoox bridges the gap between code and documentation, making it easy to keep your docs in sync with your codebase while producing beautiful, professional documents.

## Features

### Core Capabilities

- **AI-Powered Writing**: Intelligent writing assistance with context-aware suggestions and content generation
- **Code-Documentation Linking**: Bidirectional links between code and documentation, keeping them synchronized
- **Extended Markdown**: Rich Markdown syntax that supports LaTeX features while remaining human-readable
- **Multi-Format Export**: Export to LaTeX, PDF, Markdown, HTML, and more with customizable templates
- **Real-Time Collaboration**: Collaborative editing with live synchronization across team members
- **Smart Chapter Numbering**: Automatic chapter and section numbering with placeholder-based references
- **Dual Input Modes**: Write in either Extended Markdown or LaTeX syntax — Zadoox handles the conversion
- **Version Control Integration**: Git integration for versioning and automatic number replacement
- **Template System**: Flexible templates for different document styles (academic, industry, code docs)
- **Semantic Code Search**: Find and link code elements intelligently using semantic search

### Platform Components

- **Web App**: React/Next.js based document editor with AI assistant, collaboration tools, and export engine
- **Cursor Extension**: TypeScript/Node extension for Cursor IDE with code analysis, doc generation, and sync capabilities
- **Backend Services**: Document management, AI service, code analysis, sync service, and export service
- **Code Analysis**: AST parsing and semantic understanding of multiple programming languages
